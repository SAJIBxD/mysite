# Astro + MDX Site

This is a static Astro site with MDX pages and shared layout/components. Content lives in MDX, with reusable header/nav/footer and the existing styles preserved.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Content locations

- Home content: [src/content/home.mdx](src/content/home.mdx)
- Pages: [src/pages](src/pages)
- Layout and components: [src/layouts](src/layouts), [src/components](src/components)
- Static assets: [public](public)

## Routes

- `/`
- `/about/`
- `/projects/`
- `/projects/insurance-cost-predictor/`
- `/notes/`
- `/notes/why-i-prefer-simple-tools/`
- `/404.html`
