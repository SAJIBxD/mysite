### pattern 4: Two Sequence

- longest common subsequence

```
def longest_substring(word1, word2):

    m = len(word1)
    n = len(word2)
    
    prev = [0] * (n+1)
    curr = [0] * (n+1)
    
    for i in range(1,m+1):
        for j in range(1,n+1):
            if word1[i-1] == word2[j-1]:
                curr[j] = prev[j-1] + 1
            else:
                curr[j] = max(prev[j], curr[j-1])
        prev = curr
        
    return curr[n]
```
