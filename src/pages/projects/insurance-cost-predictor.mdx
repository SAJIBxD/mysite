<article class="space-y-6">
	<p>
		Insurance Cost Predictor is a small ML app that estimates insurance
		costs from common demographic and health-related inputs. The goal
		is clarity and learning: understand which features move the price
		the most, and how a basic regression pipeline behaves end-to-end.
	</p>

	<p>
		The model is trained with a clean preprocessing pipeline
		(categorical encoding + numeric scaling) and evaluated using standard
		regression metrics. The UI is deployed as a simple web app so anyone
		can try it without installing anything.
	</p>
